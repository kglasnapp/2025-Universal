# Driver - Game Pad
* In tank mode, the right joystick y controls speed of right wheels, the left joystick y controls the speed of the left wheel
* In arcade mode, the joystick y axis controls the forward and reverse motion of the robot. The x axis (side to side) deflection to cause the robot to turn right or left
* Left Bumpter -- Drive Stright must be held 
* Right Bumper - Shooter for robots that shoot
* POV up - shooter off,  right - shooter slow speed, down - shooter medium speed,  left - shooter high speed
* Y - Zero the yaw
* X - Pickup for robot that pickups objects
* A - Raise shooter 
* B - Lower shooter

# Smart dash board
* Shows various status for the robot
  * Currents for the various motors
  * Status of limit switches
  * Drive mode - mild tank, agreesive tank, mild arcade, agreesive arcade, mild curvature, agreesive curvature
